1.apply、call 有什么作用，什么区别

  • 作用
    call()方法在使用一个指定的this值和若干个指定的参数值的前提下调用某个函数或方法.
    apply() 方法在指定 this值和参数（参数以数组或类数组对象的形式存在）的情况下调用某个函数。
  • 区别
    call()方法接受的是一个参数列表，apply()方法接受的是包含多个参数的数组（或类数组对象）

原型链（prototype chain）：在 javaScript 中，每个对象都有一个指向它的原型（prototype）对象的内部链接。这个原型对象又有自己的原型，直到某个对象的原型为 null 为止（也就是不再有原型指向），组成这条链的最后一环。这种一级一级的链结构就称为原型链（prototype chain）。

继承相关问题

继承有什么作用?

  • 作用：
    1.子类拥有父类的属性和方法，不需要重复写代码，修改时也只需修改一份代码
    2.可以重写和扩展父类的属性和代码，又不影响父类本身

下面两种写法有什么区别？

  • 方法1里的printName方法是定义在构造函数内部，会在它的每一个实例上都克隆这个方法；
  • 方法2的printName方法绑定在父类Person的prototype属性下，会让它的所有实例都共享这个方法，但是不会在每个实例的内部重新定义这个方法

Object.create 有什么作用？兼容性如何？

  • 作用：
    Object.create(proto, [ propertiesObject ])方法创建一个拥有指定原型和若干个指定属性的对象。
    proto,一个对象，作为新创建对象的原型。或者为 null。
    propertiesObject,可选。该参数对象是一组属性与值，该对象的属性名称将是新创建的对象的属性名称，值是属性描述符（这些属性描述符的结构与Object.defineProperties() 的第二个参数一样）。注意：该参数对象不能是 undefined ，另外只有该对象中自身拥有的可枚举的属性才有效，也就是说该对象的原型链上属性是无效的。

参考：Object.create
